## 🛠 Tecnologias:

- HTML
- CSS
- Bootstrap
- Javascript
- NodeJs
- Handlebars
- SQLite
- Git e Github

## 💻 Para rodar:
- Clone o repositório (git clone ou download)
- Abra a pasta com cmd (ou algum prompt) e rode o comando "npm run dev" (é necessário ter o nodejs instalado)
- A aplicação ira rodar na porta 3000 (localhost:3000)
